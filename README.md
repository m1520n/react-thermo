> React Thermometer Component for React Apps

![React Thermo](https://github.com/m1520n/react-thermo/thermo.png)

## Install

```sh
yarn add react-thermo
```

## Usage

```jsx
import Thermo from 'react-thermo'

  <Thermo
    min={-50}
    max={50}
    temperature={25}
  >
```
